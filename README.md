# Presentations

RevealJS slide decks. Each presentation lives in its own subfolder.

## Viewing a Presentation

Open any `index.html` in a browser — no server required.

```
my-deck/index.html   →  open in browser
```

**Controls:** Arrow keys / Space to advance · `F` fullscreen · `S` speaker notes · `O` slide overview · `?` all shortcuts

## Creating a New Deck

```
/create-deck
```

This skill interviews you about your presentation and generates the deck.

## Themes

Available RevealJS themes: `black` (default) · `white` · `moon` · `dracula` · `league` · `beige` · `sky` · `night` · `serif` · `simple` · `solarized`

Change the theme by swapping the theme CSS link in `index.html`:
```html
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/reveal.js@5/dist/theme/moon.css">
```
